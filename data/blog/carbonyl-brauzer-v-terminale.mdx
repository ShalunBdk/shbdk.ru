При запуске веб приложений обычно их первоначальная настройка производится в браузере. Иногда разработчики привязывают настройку исключительно к локальному браузеру (настройка с адреса localhost).
Но как быть если веб-сервис запущен на виртуальной машине без GUI?

[Carbonyl](https://github.com/fathyb/carbonyl) — браузер на базе Chromium, созданный для работы в терминале.

Он поддерживает практически все веб-API, включая WebGL, WebGPU, воспроизведение аудио и видео, анимацию и т. д.

Он быстрый, запускается менее чем за секунду, работает в 60 FPS и в простое использует 0% ЦП. Он не требует оконного сервера (т.е. работает в консоли безопасного режима) и даже работает через SSH.

Для установки введите команду

```bash
npm install --global carbonyl
```

Чтобы перейти на нужный сайт например `youtube.com` введите
```
carbonyl https://youtube.com
```
